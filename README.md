# Network Packet Analyzer

A powerful Python-based network packet analyzer (similar to Wireshark) with GUI.

## Features

- Live Packet Capture from network interfaces
- Protocol Dissection (TCP, UDP, ICMP, HTTP, DNS, ARP)
- Beautiful GUI with color coding
- Packet Filtering
- Export to PCAP
- Hexadecimal Viewer
- Real-time statistics

## Installation

```bash
git clone https://github.com/467gunna/network-packet-analyzer.git
cd network-packet-analyzer
pip install -r requirements.txt
sudo python packet_analyzer.py
```

## Requirements

- Python 3.7+
- Scapy
- Tkinter
- Admin/root privileges
